TraceView
---------

TraceView program is designed for parsing and viewing trace files generated by RadixWare platform applications.

Overview
--------

The program can work in two modes:
	#Trace viewing mode.
		The trace is viewed in the window mode. A large set of filters is available. 
	#Trace files processing mode. 
		In this mode, it is possible to extract records from multiple files by some criteria (e.g. message content) and optionally split results to different files by another criteria (e.g. date).


To see help message use following command: java -jar TraceView-all.jar -h
